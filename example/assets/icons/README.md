# Icons Directory

Bu klasöre PNG ikon dosyalarınızı ekleyebilirsiniz.

Örnek kullanım:
- `custom_icon.png` dosyasını buraya ekleyin
- `button_demo_page.dart` dosyasında `PlatformIcon.asset('assets/icons/custom_icon.png', ...)` şeklinde kullanabilirsiniz

Not: iOS'ta asset dosyalarının yüklenmesi için `pubspec.yaml` dosyasında assets tanımlı olmalıdır.
